# dotfiles

There are many like it, but these ones are mine

## Get started
Clone the repository:
```sh
git clone https://github.com/marcelblijleven/dotfiles-personal.git ~/.dotfiles
# or
git clone git@github.com:marcelblijleven/dotfiles-personal.git ~/.dotfiles
```

To initialize your dotfiles, run:
```
./initialize.sh
```
